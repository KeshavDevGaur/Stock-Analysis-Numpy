# Stock Market Analysis using NumPy

This project analyzes stock price data using Python and NumPy to understand key financial concepts like returns, risk, and portfolio performance.

## Features
- Daily Returns Calculation
- Average Return per Stock
- Volatility (Risk Analysis)
- Best & Worst Stock Identification
- Highest Gain Day Detection
- Moving Averages
- Cumulative Returns
- Portfolio Analysis (Equal & Weighted)
- Sharpe Ratio
- Correlation Matrix
- Maximum Drawdown
- Z-Score (Anomaly Detection)
- Simple Price Prediction

## Tech Stack
- Python
- NumPy
- Jupyter Notebook
